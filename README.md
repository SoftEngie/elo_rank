# ELO Rank

## Install

```bash
git clone git@github.com:Tazoeur/elo_rank.git
cd elo_rank
python3.10 -m virtualenv venv
source venv/bin/activate
pip install .
```

## Launch in debug mode
```bash
launch_elo
```
